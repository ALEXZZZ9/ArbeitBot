Job creation logic:

1. askForNewJobLanguage — brings keyboard with languages to user, saves 'inputLanguageState' as user's state
2. eventEmitter.on(strings.inputLanguageState) — calls startJobDraft if language is valid
3. startJobDraft — creates job draft and saves it to user, calls askForNewJobCategory
4. askForNewJobCategory — switches user's state to inputCategoryNameState, brings keyboard with available job categories to user
5. eventEmitter.on(strings.inputCategoryNameState) — saves job's category and calls askForNewJobPriceRange
6. askForNewJobPriceRange — saves user's input state to inputHourlyRateState, sends a keyboard with available hourly rates
7. eventEmitter.on(strings.inputHourlyRateState) — calls addHourlyRateToJobDraft
8. addHourlyRateToJobDraft — sets job draft's hourly rate and calls askForNewJobDescription
9. askForNewJobDescription - saves user state to inputJobDescriptionState and adds message with inline button to cancel job description
10. eventEmitter.on(strings.inputJobDescriptionState) — calls addDescriptionToJobDraft
11. addDescriptionToJobDraft — adds description to job draft, sets user state to undefined, adds job to user's list of jobs and calls sendJobCreatedMessage
12. sendJobCreatedMessage — sends job created message and job's message with available freelancers
